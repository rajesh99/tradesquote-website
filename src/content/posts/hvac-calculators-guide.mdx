## FAQ

### Are these hvac calculators free?

Yes. Every calculator on TradesQuote is free with no signup. They produce ballpark results for planning and field checks — not stamped engineering documents.

### Which calculator should I use to size central AC?

Start with the BTU or tonnage calculator for a quick estimate. For whole-home central systems, use the Manual J calculator or hire a pro for a stamped load calc.

### What is the difference between an hvac system calculator and a room calculator?

A room calculator sizes one space. An **hvac system calculator** (Manual J) sums the whole home — accounting for duct losses, infiltration, and orientation across all rooms.

### Do calculators replace Manual J software?

Our Manual J calculator covers residential ballparks. Wrightsoft, Elite, and CoolCalc add envelope detail, report formatting, and code compliance documentation for permit submissions.
