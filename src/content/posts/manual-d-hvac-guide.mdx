**Manual D** is ACCA's standard for residential duct design. Searchers looking for **manual d hvac**, **wrightsoft manual d**, or **hvac manual d calculation** need the duct-sizing companion to Manual J — load calc tells you how much heating and cooling; Manual D tells you how to deliver it.

## Manual J vs Manual D

| Standard | Purpose | Output |
|---|---|---|
| Manual J | Load calculation | BTU/hr heating and cooling load |
| Manual D | Duct design | Duct sizes, CFM per room, friction rate |
| Manual S | Equipment selection | Match equipment to Manual J load |

You cannot size ducts correctly without knowing system CFM and available blower static pressure — both come from equipment selection after Manual J.

## Manual D Calculation Steps

1. **Determine total system CFM** — typically 400 CFM per ton (check equipment specs)
2. **Allocate CFM to each room** — proportional to room load from Manual J
3. **Calculate available static pressure** — blower rating minus coil, filter, and grille losses
4. **Size the longest run first** — trunk and branch to the farthest supply
5. **Pick friction rate** — divide available static across longest run length
6. **Size each duct segment** — ductulator lookup at chosen friction rate
7. **Verify velocity** — branch runs under ~900 fpm, trunks under ~700 fpm
8. **Size returns** — 100–125% of supply CFM, lower velocity targets

## Common Manual D Mistakes

- **Skipping load calc** — duct CFM allocation needs per-room loads from Manual J
- **Using one friction rate for all runs** — longest run sets the design rate; shorter runs may need dampers
- **Undersizing returns** — restrictive returns raise static pressure and kill airflow
- **Ignoring fitting losses** — equivalent length of elbows and tees adds effective duct length

## FAQ

### What is Manual D in HVAC?

ACCA Manual D — the residential duct design standard. It sizes ductwork to deliver Manual J loads at acceptable velocity and static pressure.

### Is Manual D required by code?

Many jurisdictions reference ACCA Manual D alongside Manual J for new construction and major renovations. Check local mechanical code.

### Wrightsoft Manual D vs free calculators

Wrightsoft handles whole-system design with fitting libraries. Free calculators size individual duct runs — sufficient for repairs and single-branch additions.

### How does hvac manual d calculation relate to a ductulator?

Manual D sets the friction rate from blower capacity and longest run. The ductulator converts CFM + friction rate into duct diameter.
